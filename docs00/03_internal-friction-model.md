# 人はなぜ動けないのか？──行動の内的要因マップ

「やりたいのに動けない」
「分かっているのに変われない」

この感覚を経験したことがない人は、おそらくいない。

そしてこの問いに対して、世の中には無数の答えがある。「意志が弱い」「習慣の問題」「環境を変えろ」「考え方を変えろ」──どれも一面では正しい。しかし、どれも全体像を捉えていない。

この記事では、人間が行動を起こす（あるいは起こせない）ときに働いている**内的要因**を、ひとつのマップとして整理する。

目的は「正解を示す」ことではない。
**「今、自分の中で何が起きているのか」を見るための地図**を提供することだ。

---

## なぜ「層」で分けるのか

人の内側で起きていることは、一枚岩ではない。

たとえば「やる気が出ない」という状態。
これは睡眠不足から来ているかもしれないし、漠然とした不安から来ているかもしれないし、「どうせ失敗する」という思考から来ているかもしれないし、「自分はそういう人間じゃない」という自己像から来ているかもしれない。

見た目は同じ「動けない」でも、原因が違えば、効く対処も違う。

睡眠不足の人に「考え方を変えよう」と言っても無駄だし、自己像の問題を抱えている人に「とりあえず寝ろ」と言っても根本は変わらない。

だから、まず**層を分ける**。

---

## 4つの層

内的要因は、大きく4つの層に分けられる。

```
　　　上
┌─────────────┐
│   自己像層   │ ← 「私とは何者か」
├─────────────┤
│    認知層    │ ← 解釈と理屈の世界
├─────────────┤
│    感情層    │ ← 反応と起動のエンジン
├─────────────┤
│    身体層    │ ← 生理的な土台
└─────────────┘
　　　下
```

| 層 | 時間スケール |
|:---|:---|
| **自己像層**（上） | 年〜十年 |
| **認知層** | 分〜年 |
| **感情層** | 秒〜時間 |
| **身体層**（下） | 分〜日 |

この記事では、**身体層を「下」、自己像層を「上」**と呼ぶ。
これは優劣ではなく、因果の流れや介入の順序を示すための表現だ。

この4層は、どこかの学派の正解ではない。
しかし、神経科学・心理学・哲学など複数の分野で独立に現れる分類が、実用上もっとも干渉が少ない形で重なる点を採っている。

### なぜこの4つなのか

層を分ける根拠は4つある。

**① 因果の向きが違う**

身体の状態は、感情・思考・自己像に強く影響する。
睡眠不足になれば、感情は荒れ、思考は短絡的になる。これはほぼ自動的に起きる。

一方、認知や自己像から身体への影響（たとえばプラシーボ効果）は、条件付きで、時間がかかり、再現性が低い。

つまり、**下から上への因果は強く即時的**、**上から下への因果は弱く条件付き**という非対称性がある。

**② 介入の「操作レバー」が違う**

| 層 | 主な介入手段 |
|:---|:---|
| 身体 | 休養・栄養・運動・呼吸 |
| 感情 | 表出・共感・安全感の確保 |
| 認知 | 再解釈・枠組みの変更 |
| 自己像 | 物語・役割の再定義 |

同じ「動けない」に見えても、どのレバーを引くかで結果はまったく変わる。

うつ状態の人に認知的アプローチ（「考え方を変えよう」）をすると悪化することがある。
一方、まず身体を整える（寝る、食べる、動く）だけで改善することも珍しくない。

**③ 時間スケールが違う**

- 身体：分〜日で変わる
- 感情：秒〜時間で変わる
- 認知：分〜年かけて変わる
- 自己像：年〜十年単位で変わる

時間軸が違うものを混ぜると、「変わらない」「変えられない」の議論が噛み合わなくなる。

**④ 崩れたときの巻き込み範囲が違う**

身体層を無視すると、感情・認知・自己像まで一斉に崩れる。
しかし認知を多少誤っても、身体がすぐ壊れるわけではない。

身体は「最下層」というより**「基盤」**。
土台としての非対称性がある。

---

## 各層の「ブレーキ」と「アクセル」

それぞれの層には、行動を止める要因（ブレーキ）と、行動を起こす要因（アクセル）がある。

---

### 身体層

#### ブレーキになるもの

- 慢性的な疲労
- 睡眠不足
- 空腹・血糖値の乱高下
- 身体の緊張（特に首、肩、腹部）

**特徴：思考以前に「やる気を作れない」状態になる。**

ここが厄介なのは、意志や価値観では突破できないこと。
「気合いでなんとかする」は、身体層の問題に対しては機能しない。むしろ精神論で押し通そうとすると、長期的に壊れる。

#### アクセルになるもの

- 適度な疲労（運動後の心地よさなど）
- 深い呼吸
- 姿勢の開放性（胸を開く、顎を上げるなど）

**特徴：理屈抜きで効く。思考より先に行動が起きる。**

身体層のアクセルは、考えなくていい。むしろ考えないほうがいい。

---

### 感情層

#### ブレーキになるもの

- 不安（予測できないことへの拒否反応）
- 恥、自己否定
- 怒りの抑圧
- 無力感

**特徴：感情そのものより、「感情を処理できない状態」がブレーキになる。**

たとえば不安。
不安があっても、それを認識して扱える人は動ける。
不安を抑圧したり、回避したりする人は、動けなくなる。

感情は「あること」が問題なのではなく、「扱えないこと」が問題になる。

#### アクセルになるもの

- 好奇心
- 微かな違和感（「なんか気になる」）
- 健全な怒り（境界を守る感覚）
- 「面白さ」

**特徴：ポジティブ感情より「探索系」の感情のほうが持続する。**

「嬉しい」「楽しい」は瞬間的には強いが、すぐ消える。
一方、「面白い」「気になる」は、じわじわと長く続く。

行動を継続させたいなら、喜びより好奇心を燃料にしたほうがいい。

---

### 認知層

#### ブレーキになるもの

- 二項対立思考（成功か失敗か、全か無か）
- 完璧主義
- 原因の内在化（「全部自分のせい」）
- 未来の過剰シミュレーション（「こうなったらどうしよう」の無限ループ）

**特徴：動かない理由が「理屈として正しく見える」。**

これが認知層ブレーキの厄介なところだ。

「リスクがあるから動かない」
「準備が整っていないから動かない」
「失敗したら取り返しがつかないから動かない」

どれも論理としては筋が通っている。だから本人も周囲も、それが「賢明な判断」だと思ってしまう。

しかし実際には、思考がブレーキとして機能しているだけのことも多い。

#### アクセルになるもの

- 仮説思考（正解探しをやめる）
- 部分最適の許容（「とりあえずここだけ」）
- 「検証する」という構え

**特徴：思考は、ブレーキにもアクセルにもなる。**

「正解を出そう」とすると止まる。
「仮説を試そう」とすると動ける。

認知層は、使い方次第で正反対の機能を持つ。

---

### 自己像層

#### ブレーキになるもの

- 固定された自己像（「私はこういう人間だ」）
- 一貫性への執着（「ブレてはいけない」）
- 他者からの期待の内在化（「〇〇さんに期待されている自分」）

**特徴：本人が「守っているつもり」でブレーキを踏んでいる。**

自己像層のブレーキは、本人には見えにくい。

「私は慎重な人間だから」と言うとき、それは自己理解のつもりで語られる。しかし実際には、その「慎重な人間」という自己像が、新しい行動を許可しないブレーキとして機能していることがある。

自分を守っているつもりで、自分を縛っている。

#### アクセルになるもの

- 「今の自分は仮の姿」という感覚
- 失敗を「履歴」として扱える（黒歴史ではなくログ）
- アイデンティティを流動的に扱える柔軟さ

**特徴：固定自己像が弱い人ほど、動ける。**

「自分はこういう人間だ」という確信が強い人は、その枠の外に出られない。
「自分は変わりうる」と思っている人は、枠自体を動かせる。

---

## 「動機があるのに動けない」のメカニズム

よくある悩みに、「やりたいことはあるのに、なぜか動けない」というものがある。

これは4層モデルで説明できる。

**欲求（やりたいこと）は、複数の層にまたがっている。**

| 欲求の種類 | 主に存在する層 |
|:---|:---|
| 空腹・睡眠・性 | 身体層 |
| 安心・つながり | 感情層 |
| 達成・理解 | 認知層 |
| 承認・意味・役割 | 自己像層 |

「やりたい」という動機の構造は、認知層や自己像層にある。
しかし、**実際に行動を起動するトリガーは、感情層に落ちる必要がある**。

つまり：

- 欲求（構造）＝ 認知・自己像に存在
- 起動（火花）＝ 感情に落ちないと発火しない

この接続が切れていると、「やりたいのに動けない」が起きる。

頭では分かっている。意味も分かっている。でも、感情が動かない。
だから身体も動かない。

---

## どの層の問題かを見分ける方法

「動けない」と感じたとき、それがどの層から来ているかを判別する方法がある。

### 方法①：問いを投げて反応を見る

「なぜ動けないのか？」と自分に問いかけたときの反応で判別できる。

| 反応 | 該当する層 |
|:---|:---|
| 理由を考えようとすると疲れる | 身体層 |
| 理由はある気がするが、霧がかかったように見えない | 感情層 |
| 理由は説明できるが、なぜか納得しきれない | 認知層 |
| 理由を疑うという発想自体が出てこない | 自己像層 |

### 方法②：介入テストで反応を見る

順番に試して、改善するかどうかを見る。

1. **寝る・食べる・身体を動かす** → 改善したら身体層
2. **誰にも評価されない場所で感情を出す** → 改善したら感情層
3. **状況を紙に書き出して仮説化する** → 改善したら認知層
4. **「自分はこういう人間だ」を疑ってみる** → 改善したら自己像層

## 介入の順序：原則と例外

### 基本原則

**迷ったら、下（身体）から始める。**

理由は単純で、下の層が整っていないと、上の層への介入が効かないから。

睡眠不足の状態で「考え方を変えよう」としても、脳がまともに動かない。
身体が緊張している状態で「感情を出そう」としても、出てこない。

だから：

- 迷ったら下から
- 詰まったら一つ上へ
- 壊れたら一つ下に戻る

### 修正原則：固い層がある場合

人によって、「どの層が固いか」は違う。

ある人は感情層が固く、ある人は自己像層が岩盤のように動かない。

**固着の定義：変化したときに、他の層へ過剰な反動を起こす状態**

たとえば、感情層が固い人に感情を扱わせようとすると、急に身体症状が出たり、認知が混乱したりする。

固い層を特定する手がかり：

| 観察される反応 | 固着している可能性が高い層 |
|:---|:---|
| 介入した直後に、別の問題が噴出する | その層 |
| 変化することを強く嫌悪・否定する | その層 |
| その話題になると逸らそうとする | その層 |
| 極端な一般化が出る（「どうせ」「絶対」） | 自己像層 |

**固い層がある場合の原則：最も固い層の「一つ下」から叩く**

固い層を直接触ると、反発が起きる。
しかし、その下の層から事実を積み上げていくと、上の層が自然と緩むことがある。

## 自分で観察できないとき

ここまでの話は、「自分で自分を観察できる」ことを前提にしている。

しかし、うつの深い状態、強いストレス下、解離状態などでは、その前提が成り立たない。

- 「試す」という発想自体が出ない
- 変化したかどうかを評価できない
- 「今、自分がおかしい」という比較軸がない

この状態では、内的フレームワークは停止している。

### 外部介入の役割

このとき、外部（他者、環境、ルール）が果たす役割は、**「メタ認知の代理」**だ。

本人が観察できないなら、外から観察する。
本人が判断できないなら、外から初期条件を整える。

| レベル | 本人の状態 | 介入の主体 | 対象 | 言葉の必要性 |
|:---|:---|:---|:---|:---|
| 0 | 自己観察が不能 | 環境・他者 | 身体のみ | 不要 |
| 1 | 感情だけ断片的に感知 | 他者 | 身体＋感情 | 短く具体的に |
| 2 | 認知が戻り始める | 本人＋他者 | 認知 | 仮説形式で |
| 3 | 自己像に触れられる | 本人 | 自己像 | ─ |

**レベル0では、本人の理解も同意も必要ない。**

強制的に寝かせる。食事を出す。危険な刺激を遮断する。
説明も説得も要らない。まず身体を守る。

この順序を崩すと、回復しても再発しやすくなる。

## 「健全」とは何か

ここまで「健全な怒り」「処理できる状態」といった表現を使ってきた。

では、何をもって「健全」とするのか。

**このフレームワークにおける「健全」の定義：**

> **他の層への侵食が最小で、切り替え可能性が残っている状態**

これは道徳的な判断ではない。システムの安定性の問題だ。

たとえば怒り。

- **健全な怒り**：表出しても、身体が壊れず、自己像も崩れない。怒った後に切り替えられる。
- **不健全な怒り**：身体症状に転化する（頭痛、胃痛）。または自己否定に変わる（「怒った自分は最低だ」）。切り替えが効かない。

感情があること自体は問題ではない。
その感情が他の層を侵食し、固着するかどうかが問題になる。

## 全体像

最後に、全体の構造を図示する。

```
┌───────────────────────────────────────────┐
│              自己像層                      │
│    許可と禁止／長期安定／「私とは何者か」    │
├───────────────────────────────────────────┤
│              認知層                        │
│    言語化／モデル化／理由の生成             │
├───────────────────────────────────────────┤
│              感情層                        │
│    起動トリガー／双方向性／抑圧の発生点     │
├───────────────────────────────────────────┤
│              身体層                        │
│    基盤／即時性／意識の介入が弱い          │
└───────────────────────────────────────────┘

        ↑ 強い因果（即時・半強制）
        ↓ 弱い因果（条件付き・遅延）

外部 ──→ 観察機能の代理 ＋ 初期条件の供給
```

---

## まとめ

- 人の内側で起きていることは、4つの層（身体・感情・認知・自己像）に分けられる
- 各層には固有のブレーキとアクセルがある
- 「動機があるのに動けない」のは、欲求の構造と起動トリガーが接続していないから
- どの層の問題かは、問いへの反応や介入テストで判別できる
- 介入は基本的に下から。固い層がある場合はその一つ下から
- 自己観察ができない状態では、外部がメタ認知を代理する
- 「健全」とは、他層への侵食が少なく、切り替え可能性が残っている状態

このマップは「正解」ではない。
しかし、「今、自分の中で何が起きているのか」を見るための、ひとつの補助線にはなるはずだ。

---

*動けないとき、まず確認すべきは「どの層が詰まっているか」。*
*そして、詰まっている層に合った介入を選ぶこと。*
*それだけで、空回りはかなり減る。*
