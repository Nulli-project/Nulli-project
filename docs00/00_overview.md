
# Nulli プロジェクト

**Nulli プロジェクト**は、著作物ではなく、自己複製する認識のシステムです。
異なる角度から見て日常の選択肢を増やす為のツールを提供します。

完成した理論を配布するのではなく、「構造を見る視点による選択肢を広げる」という種を共有する試みです。
その種を受け取った人が、共鳴し、それぞれの視点で感じている事を掘り下げ、可視化し、発信する。
その情報がまた別の誰かの種になる——この循環が、プロジェクトの核心です。

### 「Nulli」とは？
Nulli = null + i = 非存在の自己
これは固有名ではなく、一人称的な普遍視点です。
重要なのは「誰が」ではなく、「何を見たか」「どう考えたか」です。
共鳴したときあなたの中にもNulliが生まれる。

このプロジェクトは完成形ではなく、進行中の探索です。
「無意識の選択」(無意識下における選択と行く末)という概念を提示することで、
実際に人々の選択への意識は変わるのだろうか？その変化が累積すると、何が生まれるのだろうか？
私自身、まだ答えを持っていません。だからこそ、これを進行形として公開します。

### 無意識の選択
朝、目覚めてスマホに手を伸ばす。この選択を365日続けると、1年後のあなたの注意力は変わっている。
階段とエレベーター。毎日エレベーターを選ぶと、5年後の体力は違っている。
人と目を合わせるか避けるか。この選択を積み重ねると、人間関係が変わっている。
日常の選択が、意図せず未来を形成している。
[無意識の選択](01-1_unconscious-choice.md)

**この構造を可視化することで：**
  -	自分が何に投票しているか気づく
  -	選択肢が拡大する
  -	変化の余地が生まれる
