 Nulli プロジェクト

Nulli プロジェクトは、著作物ではなく、自己複製する認識のシステムです。
そして認識の自己複製システム 

異なる角度から見て日常の選択肢を増やす為のツールを提供します。

完成した理論を配布するのではなく、「構造を見る視点による選択肢を広げる」という種を共有する試みです。
その種を受け取った人が、共鳴し、それぞれの視点で感じている事を掘り下げ、可視化し、発信する。その情報がまた別の誰かの種になる——この循環が、プロジェクトの核心です。

理念
社会の長期的存続には「全員が少しだけ協力する構造」と、
その基盤となる「個々人の行動に対する自覚」が必要である。
したがって本プロジェクトは、
（A）個人が“自分が何をしているか”を直観的に理解できる可視化、
（B）個人の選択肢を現実的に広げるための情報・ツール配備、
（C）共鳴した参加者による多視点の分析共有を一本に繋ぐ「思考と行動のための装置」を目指す。
原則：非強制・非押付け、バランス原理（メリット⇄デメリットを常に対で提示）、
